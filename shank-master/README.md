# shank

******使用指南******
一、软件功能：
欢迎使用Shank约车应用~
1、座位有20个，超过不能预约。
2、一个用户不能重复预约。
3、用户可以取消预约。
4、了解更多按钮里可以查看行车安排和北大新闻。

二、系统部署：
1、将完整的Shank文件夹下载至本地。
2、如果改变部署的微信端服务器地址，就要改变/routes/index.js里的AESkey和token信息，有自己定。
3、首次运行有可能需要修改access_token,因为post菜单需要使用到。程序中没有自动修改的途径，需要手动（会在命令行里打印出来）复制到代码里，在post_str里。
4、就可以直接运行，node index.js
