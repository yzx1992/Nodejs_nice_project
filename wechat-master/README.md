# wechat （微信公众号）历史上的今天 
##1.软件介绍
  此微信公众号的功能是当你输入指定的文本后，向用
  户推送历史上的今天发生的重大事件。大致流程如下：
  根据每天的日期，到响应的网站上爬取内容，然后根
  据用户发送的消息向用户返回爬取的数据内容。软件
  包括两个模块文件，history.js是用来爬取网站数据，
  index.js是主程序文件，用来接收用户信息，以及向
  用户返回爬取的内容。
##2.系统部署
  本系统基于nodejs环境开发，在部署有nodejs的环境
  下，执行 node index.js运行。
